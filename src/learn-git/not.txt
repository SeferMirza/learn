notlar:
 git push --set-upstream origin branchTest  -> yeni branch, origin bağlantısına gönderilir.
 use "git restore <file>..." to discard changes in working directory
 git merge <branch-name> -> branch te yaptığın değişiklikleri commit attıktan sonra master dalına geçerek merge yapılır.
 *eğer git log yazarak geçmişe bakıyorsan 'Q' ile geçmişten çıkabilirsin :D
 git branch -D <branch-name> -> delete local branch
 git push origin --delete branchTest -> delete remote branch
 
sorun var:
 eğer sen başka bir dalda master dalından öndeysen ve commit ten sonra master dalına geçersen master da commit atman gerekiyor.
